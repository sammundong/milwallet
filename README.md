# 밀월렛 - 군인 자기계발 플랫폼

군 장병을 위한 자기계발 및 제휴 커머스 플랫폼

## 주요 기능

### 1. 헬스 카테고리
- **맞춤 루틴 제공**: 영내 맨몸운동, 벌크업, 특급전사 도전 루틴
- **체력검정 트래커**: 팔굽혀펴기, 윗몸일으키기, 3km 달리기 기록 관리
- **운동 보조제 & 용품 추천**: 크레아틴, 유청단백질, 저항밴드 등
- **실시간 루틴 가이드**: 단계별 운동 안내 및 완료 후 상품 추천

### 2. 건강(영양제) 카테고리
- **건강 목표별 추천**: 피로, 활력, 근육, 면역력, 수면 등
- **기초 필수 영양제**: 종합비타민, 오메가3, 비타민D3+K2
- **기능별 영양제**: 홍삼, 마그네슘, L-테아닌, 루테인
- **군 급식 활용 가이드**: 단백질 섭취 전략, 체중 관리 식단

### 3. 도서 카테고리
- **카테고리별 추천**: 자기계발, 재테크, 어학·시험, 인문·교양
- **군생활 맞춤 이유**: 각 도서가 왜 군 복무 중 읽기 좋은지 설명
- **독서 챌린지**: 월별 테마 독서 목표 설정
- **독서 기록 관리**: 완독/읽는 중 구분, 한줄평 및 핵심 문장 기록

### 4. 군수(군대수능) 카테고리
- **목표별 학습**: 수능 재도전, 편입 준비, 공무원 시험, 어학 시험
- **과목별 학습 트래커**: 국어, 수학, 영어 학습 시간 및 점수 관리
- **추천 교재**: EBS 수능완성, 마더텅 기출, 편입 영어
- **온라인 강의 안내**: 메가스터디, 이투스, EBSi 군인 할인 정보
- **12개월 로드맵**: 복무 기간에 맞춘 수능 준비 전략

### 5. 자격증 카테고리
- **IT 자격증**: 정보처리기사, SQLD, 컴퓨터활용능력, AWS
- **어학 자격증**: TOEIC, JLPT, HSK, OPIc
- **금융 자격증**: 한국사능력검정, 펀드투자권유대행인
- **합격 루틴**: 90일/60일/30일 단계별 학습 플랜
- **복무 중 합격 가능 여부**: 각 자격증별 난이도 및 준비 기간 안내

### 6. 피부 카테고리
- **피부 타입 진단**: 건성, 지성, 복합성, 민감성, 정상
- **피부 고민별 솔루션**: 여드름, 트러블, 건조함, 칙칙함, 모공, 주름
- **아침/저녁 루틴**: 단계별 스킨케어 가이드
- **추천 제품**: 세타필, 라운드랩, 닥터자르트, 메디힐
- **PX 대안 제품**: PX에서 구매 가능한 대체품 안내
- **단계별 체크리스트**: 루틴 시작 시 각 단계 완료 체크

## 수익 모델

### CPA 기반 제휴 수수료 구조
- **운동용품**: 4% 수수료
- **영양제**: 3% 수수료
- **도서**: 3% 수수료
- **온라인 강의**: 5% 수수료
- **화장품**: 4% 수수료

### 네이티브 광고 방식
- 배너 광고 없이 콘텐츠에 자연스럽게 통합
- 루틴 완료 후 맞춤 상품 추천
- 커뮤니티 후기와 상품 연결

### 예상 수익 (MAU 1만명 기준)
- 영양제: 월 약 79만원 (전환율 12%)
- 도서: 월 약 60만원 (전환율 8%)
- 화장품: 월 약 108만원 (전환율 15%)
- 온라인 강의: 월 약 267만원 (전환율 6%)
- **총 예상 월 수익**: 약 633만원

## 핵심 차별화 포인트

1. **루틴 기반 추천**: 운동/독서/스킨케어 루틴 실행 중 자연스러운 상품 노출
2. **군생활 최적화**: PX 구매 가능 여부, 군 도서관 재고 확인 안내
3. **커뮤니티 연동**: 실제 장병 후기와 상품 구매 연결
4. **절약 가시화**: 제휴 구매를 통한 누적 절약 금액 표시
5. **성장 대시보드**: 월별 자기계발 활동 통합 요약

## 기술 스택

- **Frontend**: React 18, Vite
- **State Management**: React Hooks (useState, useEffect)
- **Storage**: localStorage (구매 이력, 루틴 기록)
- **Deployment**: Vercel
